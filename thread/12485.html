Duke 20:53
go to here
http://renjuoffline.com/get_games.php
download Current Games XML
download xml2bdt.rar
unpack all
then run
xml2bdt.bat black=warpod
file games.xml.bdt will contain games of nickname "warpod" as black
rename games.xml.bdt to player_black.bdt
xml2bdt.bat white=warpod
rename games.xml.bdt to player_white.bdt
those two files now has only games by player warpod
Run RenLib and open those games
use needed nick instead of "warpod"
