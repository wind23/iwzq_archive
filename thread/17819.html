#1 西安一些比较爱违章的地段点评 作者：悟石 发表时间：2012-5-16 17:12:25

把西安一些比较爱违章的地段给大家说一说：
城东
     1、长乐坡东郊车管所门口有测速器和摄像头由东向西行驶车辆注意。
     2、长乐坡铁桥东侧路口红绿灯处有球形摄像头。
     3、长乐坡十字遇红灯时停车不能超白线。
     4、半坡十字有测速器（由西向东）和摄像头（东西双向）。
     5、交通大学北门口（兴庆公园正门）以东装有摄像头监视，兴庆路由东向西行驶压黄线 。
     6、咸宁东路浐河桥西，有测速装置和摄像头，限速50 km。
     7、咸宁东路红旗市场十字有球形摄像头。  
     8、咸宁中路83中门口，新装4个摄像头。
     9、兴庆路咸宁路路口北侧，南北方向的车道上有测速，40公里很容易超的。
    10、豁口十子向西有JC查压黄线 。
    11、西京医院门口左传时一定要转大些,要不然会被照.说逆行嘞!
    12、兴庆路西京医院门前右转车辆要走原来的自行车道,原右转道改为左转和直行了。
    13、在东二环北头双向，北二环东头双向，两个距转盘都是三五百米，有测速摄像头。
城南
    1、电子正街全段禁停，连马路牙上都不能停，每天都有JC贴黄条，是高速支队的。
    2、电子城一带摄像头很多，谨慎驾驶。高新路和光华路十字创灯压线中招率极高。
    3、太白路南段快到绕城高速入口路段,路边是不能停车的,高速大队管辖范围。
    4、太白南路与电子商城丁字路口查安全带。
    5、南二环太白南路立交桥南约50米有摄像头，别在那里掉头。 
    6、太白南路与光华路（西安科技大学十字东与十字北）有摄像头查压黄线。
    7、石油学院南门口有摄像头，那条东西路由东向西,在623所附近多JC严罚走飞机动车道。
    8、南捎门十字 --压白线 ，体育北路---压黄线, 南稍门十字的摄像头查穿插行驶,大家注意不要轻易变线，百佳汽贸的青龙路上有摄像头查压黄线。
    9、明德门杨家村十字北侧的摄像头凶猛异常，专逮违章超车和压黄线。
   10、科技路装有测速探头。
   11、西万路与丈八路路口有JC检查安全带，注意系上安全带。
   12、雁塔路向建设西路、东路严禁左拐。
   13、纬二街全段有摄像头及测速器，严禁压黄线及超速。
   14、翠华路财院门口树后面有摄像头、监视压黄线。
   15、小寨飞旋广场北侧,也就是金壁辉煌那一侧千万不要停车,经常见交警贴条。
   16、西万路－科技路十字北，有一组四个摄像头，监视十字以北压单黄线。
   17、友谊西路黄雁村十字向西有4个摄像头查省医院北门口违章掉头，需掉头车辆可前行约30米路口可掉头。
   18、和平门十字尤其是从东向南拐，再挤都不要跟警察的手势，压了黄线，只有压线镜头，没有警察手势，惨啊！
   19、还有唐延路限速40 km。
   20、电子城也属高速支队，一定要注意。
   21、连高速路收费站的录像也算电子警察的范围  （皇帝手植柏） 那条路的确是限速的,还有查安全带的,不过是随机的。
   22、另外,长安区大学城前面的道路拆了限速杠以后限速60 km,几乎每天都有摄相的车蹲       在慢车道,抓到了200。
   23、外院到雁塔医院有视频照安全带。
   24、大雁塔南至水厂路经常有十一大队巡逻车查超速，限速50。
   25、太白路限速60 km，部分路段限速50 km。
   26、还有电子二路的南侧不可以停车,高速大队每天拍摄,连台子上面包括洗车行里都不行,        要停就到北侧。
   27、西安至沣峪口路段限速60，属长安交警队管，要小心。
   28、子午大道被摄，警察说超速50 %，罚款1000大元，扣6分。
   29、市**小区南崇业路、永松路两边禁止停车，有JC贴黄条。
   30、太白小区十字新加了四个摄像头。
   31、兴善西街停车在北边、兴善东街停车在南边或指定停车场停。否则贴黄条。
   32、南郊电视塔西有测速的摄像头!
   33、西二环与南二环拐弯偏东处(高新四路西)新增加测速摄像头(限速60)，监视由西向东车辆的情况。
 城西
    1、桃源路，由南向北，过省**小区，有摄像头拍摄压单黄线 。
    2、近期高速交警大队的雷达测速车在高新区几条道路上进行不定时、不定点的流动测速，为了各位的安全和经济不受损失，请大家注意限速标志。  
3、西部大道全线限速50 km。
    4、博士路限速20 km。 
    5、西沣路限速60 km。
    6、环西路西门什字－－－玉祥门盘道路段，过了立交，千万不要在左拐道上直行－－－会被照的。
    7、劳动南路和西稍门什字北边至第一个三叉路口中间有4个摄像头检查压双黄线；十字西边10米处有摄像头摄禁停。
    8、西郊枣圆西路东西双向都有测速摄像头。
    9、刚开通的西二环大兴路立交上有摄像头，请注意。
   10、西二环和昆明路十字向西一百米，向西有测速摄像头！向东有摄像头查压黄线！。
   11、西二环土门立交南新增加测速摄像头(限速60)，监视由南向北车辆的情况。
城北
     1、北二环文景路口既抓逮违章掉头也抓超速 。
     2、在经济开发区的凤城一路,凤城二路,凤城三路,凤城四路,和凤城五路路边还有道沿上不许停车!八大队警察开的车录象，有次,我停车在路边,被录象，罚200。
     3、去机场高速的最后一个红绿灯十字，一定要把车速控制在60 km，在十字路口测速，在第二个转盘就有交警等你。
    4、北二环西段(白杨东新村)新装测速摄像头两套，可监视向东或向西两个方向的车辆行驶情况。
 城内 
     1、南大街北行,走钟楼盘道的慢车道,罚100米。 
     2、粉巷内，特别是王子饭店前千万别越线行驶 。
     3、尚朴路全段禁止停车。 
     4、东新街：尚德路什子，西向北拐，小心压黄线。老有一个头上有摄像头的警车藏在停车场里偷拍 。
     5、西新街：解放路口到尚德路口的中间（东向西方向）经常有车掉头，4大队的警察乐此不疲地在这里开罚 。
     6、过和平门十字千万要注意，不要跟着前车提前转弯，和平门外路西口上是个球型摄像头， 拍的很清楚。提前拐弯压黄线了，在一大队处理，扣2分，罚100米 
     7、火车站：由东向西从东引桥进入东广场时,在引桥上不要从右侧超车；由西广场向西行，站前分局JC在路边小店里摄象。北门过来车辆也不可在南侧停留，JC在一汽车调度亭摄象。（JC在的地方都有茶色玻璃，司机根本看不到）。
     8、环城南路从文艺路北口至南门段由西向东单行  。
     9、西大街城隍附近街道的路北严禁停车，没有禁停标志，但是，8大队经常贴黄条子。 建议大家把车车停在马路的南面。
    10、尚德门可出不可进，注意标志 。
    11、北大街什子向东至省**北门不要掉头，五大队暗摄像，罚100米 
    12、建国门内禁止停车。 
    13、还有朱雀门十子环城路上，摄像头多成渠列，去年被拍2次。
    14、大车家巷违禁停车，我就被罚过，看着车家巷停了那么多车，可是大家千万要小心，三中队经常在那里照相，还有南大街由南向北，车堵的时候千万别走慢车道，绝对会被抓的 。
    15、西北三路十子的摄像头可以拍摄到车内的驾驶员有没有系安全带。
    16、莲湖路快到北大街十子向东可以拍摄到机动车走慢行道 。
    17、五味什字，六中门口附近，不能停车，贴黄条，3大队。
    18、玄风桥东口 一大队经常在这里守侯抓逆行单行道。
    19、西华门十字向西无论掉头还是行驶，必须安指示牌行驶。十字向西架有摄像头！很厉害！
