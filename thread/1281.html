如何下好开局

五子棋教室

#1 如何下好开局 作者：有志青年 发表时间：2006-11-15 17:38:51

一些开局中，那些类似迷宫般的变换可能会令学习者迷茫，即使专论开局的作家也会被找出破绽。我写过一些书，读者同样也能指出谬误！没必要要求作者指出开局中每种可能的变换，尤其是在那些非强制性行棋的开局中，出现不同的顺序是很平常的，不过以不一般的方式获得同一局面的情况有时确实需要特别指出。读者尤其感到气恼的是那些在同一本书的不同章节分析的局面，却用不同的例子并作出不同的评价结论！就算在著名的 “ 开局百科 ” 中也找得出这种错误的例子。
　　在行棋次序和变例中施诡计是很多棋手武器库中的一大组成部分。现在的对局，对手的和自己的，一经完成就公布，而彼此水平相近。于是，高手总要掘地三尺发现细微差别，在航行在自己熟练变化的同时避开对手所熟练的变化。他们努力置对手于不利或不熟悉的变化，如果做不到，那就至少努力让对手偏离他们最想得到的变化。

　　任何人只要经常下棋，都会逐渐开发出一种开局准备，不管是偶发的 ( 象多数业余者 ) 还是精心布置成变化之间紧密相连的 ( 高手 ) 。这样的开局准备建筑在你因感到自在自信而乐于采用的局面之上，不过得获取了那样的局面是才有得说，因此在打造自己的开局准备时你既要考虑自己喜欢的变化，又要考虑获取它们的最佳路线。
　　在向所喜欢的变化进发时，如果问题出在主要行棋路线上，你可以试试变换，换其它更可靠的行棋次序获取同样的局面。你所走的每一步棋都是过程，而随之出现的每个局面对手都要对付。着法本身足以向对手造成心理影响的很少，不过出现的局面本身才是真实有效的。
