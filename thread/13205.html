function Add_Win($dr,$w)
{
    return round(32*($w-(1/(pow(10,(-$dr/400))+1))),1);
}

浙江慢棋的计算方法：
先计算 10的（-delta/400）次方，其中delta是双方分差，比如分差20分，则结果为10^（-10/400）=9.975

然后+1 就是9.975+1 = 10.975

然后1/10.975 = 0.0911

然后1-0.0911 = 0.9089

32* 0.9089 取整，得29

蓝色背景的引用是等级分预期得分率的计算公式，预期得分率=1/[10^（分差/400）+1]，其中分差=对手积分-自己积分
对局结果按照胜1、和0.5、负0计算，积分变动值（四舍五入，保留整数）=K值×（对局结果-预期得分率），这里的32就是K值
只是可惜楼上计算有误，10^（-10/400）=0.944≠9.975，后续计算预期得分率=1/(0.994+1)=0.514，最后计算胜者得分=32×（1-0.514）=15.552。
