#1 RenlibPPC 1.1[用于POCKET PC 的五子棋打谱软件] 作者：有志青年 发表时间：2006-10-16 17:13:23

本软件是为五子棋爱好者量身定做的，用于POCKET PC 的打谱软件。
该软件参照 WINDOWS 桌面版 RENLIB  ，在功能上也尽力向 RENLIB 靠拢

使用方法
1。解压缩；
2。将解压缩后的文件夹复制到 PPC 上任意目录即可（目录路径最好不要包含中文），WINDOWS DLL 目录可不复制过去；
3。执行 RenlibPPC.exe；
4。WINDOWS 文件夹里面的文件为普通 PC 上运行的程序。
5。尽可能将文件放置于内存，操作速度会明显加快。

功能列表：
1。棋谱打开和保存；
2。编辑棋谱：删除添加棋子，添加编辑评注；
3。棋谱支持树状结构；
4。自动保存棋谱
5。棋谱文件为 sqlite3 数据库文件，使用方便，便于其他软件或网站读取数据；
6。导航功能：第一手、最后一手、上一手、下一手；
7。显示/隐藏数字；
8。察看模式/遍及模式
9。研究功能（将当前变化导入研究用棋盘进行研究，而不会更改棋谱数据）
10。搜索评注；
11。同一软件可同时用于WINDOWS 操作系统 、  POCKET PC  和 WINDOWS 职能手机；
12。方向键：向上（前一手），向下（后一手），向左（最先手），向右（最后手）。
13。工具栏使用说明：
 |<  至开始
 < 上一手
 > 下一手
 >| 最后手
 N 显示/隐藏数字
 D 删除当前点
 E 是否为编辑模式
14。棋谱保存为图片

注意事项：
1。系统要求：.NET 精简框架 1.1 以上。
2。若要在 WINDOWS 操作系统中使用，请先在WINDOWS 上安装 .NET FRAMEWORK 1.1以上，或者 .NET 精简版 1.1 以上。
