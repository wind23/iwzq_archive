参加2011年五子棋世界通讯锦标赛！
五子棋朋友们你们好！
我现在来介绍一下Taraguchi2011世界五子棋通讯赛！
2011世界五子棋通讯赛将分为三个团组进行：

高水平组(AR组,10名选手,RIF规则)，第一团组(B组,C组,每组12人,Taraguchi规则)，第二团组(GroupsD,E,F,G,TaraguchiRule)。第二团组的大小根据报名人数而定。

由于比赛规则有变，AR比赛对之后高水平组的比赛不会有影响。但是AR的选手可以报名参加B，C组的比赛。

B，C组的前8名棋手（即每组前4）和2名通讯赛得分最高的棋手，第二年取代原10名棋手升入高水平组比赛（Taraguchi规则）。
8名D-G组的棋手（小组前2）第二年升入第一团组比赛，原B，C组的后8名（每组后4名）被降级。

本比赛在五子棋离线服务器（RenjuOfflineServer）上进行(http://renjuoffline.com)
请注意，本服务器不支持通过e-mail下棋，只能通过本界面进行。
开局规则AR组是RIF标准开局，B,C,D及其它组是Taraguchi开局。去年在高水平组使用的是RIF规则。

时间：每名棋手所有棋局135天，如对手为同一个国家，每人只有115天。每手棋21天（此为服务器规则，到时，服务器自动执行裁决）。每局棋超过150子则判双方和棋。
如果一名棋手放弃的棋局数量太多，他可能会被取消资格，他的所有棋局均被判负。裁决由他所在的相应组别的裁判作出。

注意：亲爱的棋手们，请相互尊重！按五子棋“公平竟争”的精神进行比赛。如主裁判确认某棋手采取“不公平”的方式（如：在明显获胜的棋局下认输或建议和棋，或不挡冲四等）来确保对手取得好成绩。裁判可以取消该棋手的比赛资格，他的所有棋局将被判负。这种情况在高水平组由裁判委员会裁定。

比赛开始时间为2011年2月1日。
报名截止日为2011年1月25日。

欢迎所有的五子棋爱好者参加！

报名信息应包括以下内容：
1. 你的全名。
2. 你在RenjuOfflineserver上的用户名。
3. 你所在的国家和城市。
4. 你的电子邮箱
5. 联系人：全名，手机和电子邮箱，这个联系人可以确认你的参加和证明你的身份。可以是你们俱乐部的领导或棋手，你的好朋友（也可以是其它国家的）。如果找不到这样的联系人，请联系裁判一起解决此问题。
6. 你的邮政地址（高水平组需要填写，其它组不需要）

请把你的申请同时发到下边两个邮箱：renjusignup@narod.ru和owene@pisem.net以确保报名成功。
每函必复。如一周内没有收到答复，请重新发送或通过其它方式联系我，如通过RenjuOffline给我发短信（我的用户名为Owen）或通过SMS与79166348954联系。

友好的，

TAKE PART IN THE WORLD CHAMPIONSHIP 2011 IN CORRESPONDENCE RENJU!

Dear renju players!

2011 year correspondence tournament will be held in 3 leagues:

High League (Group AR, 10 participants, RIF rule), First League (Groups B, C, 12 participants each, Taraguchi Rule), Second League (Groups D, E, F, G, Taraguchi Rule). The size of the Second League is an approximation and will be finally set before the beginning of the tournament.

The AR tournament will not affect the participation in the next High League because of the changing of rules. But it is permitted for AR participants to sign up and play in groups B, C.

8 top players from Groups B and C (4 players from each) and 2 players with the highest correspondence rating will play in the High League next year instead of 10 players of this High League.

8 top players from Groups D-G (2 players from each) will play in the First League next year instead of 8 bottom players from Groups B and C (4 players from each).

The tournament is held at Renju Offline Server (http://renjuoffline.com).
Please note that this server doesn't support playing via e-mails, only from the interface.
The opening rule is RIF standart rule for AR group, Taraguchi certified rule for groups B, C, D and others.

Time control - 135 days to each player for all game, and 115 days if both players represent the same country. Besides that, there is an additional rule used at the server - 21 days per 1 move (this rule is only server's rule, therefore referee does not stop games by this rule; but the server does!). After 150 moves a game is declared as a draw.
