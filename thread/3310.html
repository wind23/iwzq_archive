（中）转战中游

知道中游，大概是 2002 年 8 月吧，是一个同事告诉我的，他说中游的台球比联众好玩，带着好奇，我下载了中游的程序，首先第一件事就是直奔五子棋棋室。中游的五子棋棋盘首先吸引了我，感觉它很有质感，然后发现中游赚分比联众容易，赢一局 5 分，输一局才扣 2 分，挺划算的，于是，我就开始在中游下棋了（非常功利吧？嘻嘻）。注册了一个叫空谷幽兰的 ID ，正式开始了我的中游之旅。

在中游下了几盘棋，居然被我全胜了，挺得意的，感觉自己也算高手了（汗）。来了一个胜率比较高的，和我下，我黑棋，双三胜了。那个人开始骂我，说我卑鄙，居然不用禁手，怪不得胜率 100 ％。五子棋有禁手？第一次听说。原来我习惯坐在桌子的右边，所以规则由我设置，我看见出来的对话框里有一个禁手，打着勾，我不知道什么禁手，下意识的取消掉了。什么是禁手？为什么不设禁手会被骂？这个问题困扰了我好久。

#5 风中棋缘（续）彷徨 作者：有志青年 发表时间：2008-1-31 17:23:34

风中棋缘（续）彷徨

再名贵的树，也不及记忆中的那一棵

潮涨汐蚀 , 日出日落，月圆月缺

曾经的人曾经的事，曾经的最爱

都已随风而逝……

重新回到了风族，坦率的讲，我的社团观有了一些变化。我不再是以前那个为社团竭尽所能的好管理员了，上网，主要是为了我前面讲的两个主题：开心下棋、开心交友。对社团的付出，比以前少了很多。因为风过无痕那个 ID 经常会有人找，所以我常常同时挂几个 ID 在中游，很少用那个 ID 下棋了。用非管理员的 ID 下棋，一般会下得更好，因为可以更专心，可以不顾忌很多东西。但是，我常常还是为自己的棋无法有进一步的提高而苦恼。心中，一直有个愿望——找一个师父教教我系统的学习一下如何下五子棋。
